Client public assets (served directly by URL):

- `/assets/Mafia-Icon.png` used by favicon and page headers.
- `/assets/icons/*` for static icon files.
- `/assets/images/*` for static images used in UI.
- `/assets/audio/*` for music/sfx loaded by URL.
- `/assets/fonts/*` for self-hosted font files.

Folder layout:

- `icons/`
- `images/`
- `audio/`
- `fonts/`

Use `public/assets` when files should be reachable as plain URLs.
