// What Goes In Here //

## src/index.ts

Server Point Entry:
    - What it does
        -- Start HTTP Server
        -- Attach Socket.io
        -- Handle New Connections
        ``` io.on("connection", socket => {
                registerSocketHandlers(socket)
            })  ```
    *** NO GAME LOGIC IN HERE ***  

## src/socketHandlers.ts

All Socket Events Live Here:
    - Contains and Handles:
        -- create_room
        -- join_room
        -- send_chat
        -- vote
        -- disconnect

    - What it does:
        -- Receives Events
        -- Validate Inputs
        -- Calls room/game Functions
    * Essentially a controller layer *

## src/room.ts

Room Lifecycle Management:
    - Contains and Handles:
        -- Creating rooms
        -- Joining rooms
        -- Removing Players
        -- Deleting Empty Room
        -- Storing Room State
        ``` const rooms: Map<string, Room> = new Map() ```
    *** FILE SHOULD NOT CARE ABOUT MAFIA RULES, ONLY ROOM STATE ****

## src/gameLogic

Actual Mafia Logic:
    - Contains:
        -- startGame.ts
        -- assignRoles.ts
        -- nextPhase.ts
        -- handleVote.ts
        -- checkWin.ts
    ** Each file does one thing **

    - Why?
        -- Easier to debug
        -- Easier to add modes later
        -- Prevents Spaghetti code
